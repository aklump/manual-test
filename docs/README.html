Quick Start

  1. composer require aklump/manual-test
  2. cp ./vendor/aklump/manual-test/examples/config.xml ./config.xml
  3. Modify config.xml as necessary
  4. Create at least one folder, with a manual test.
  5. Relative paths are relative to the config file.
  6. export TEST_BASE_URL=http://mysite.com; ./vendor/bin/generate --configuration=config.xml
